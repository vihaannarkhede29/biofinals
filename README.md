# Biology Honors Mock Final Exam

This repository contains a **mock final exam** created for a Biology Honors class to help students prepare for the real final. The goal of this project is to make studying more engaging and interactive than traditional note review by allowing students to test their understanding and identify areas that need improvement.[web:2]

The mock exam is built in **HTML** and deployed at:  
**Live site:** https://biofinals.vercel.app/ [web:1]  
This makes it easy for anyone to open the exam directly in a browser without cloning the code.

## Features

- Multiple-choice questions covering key Biology Honors topics (cell biology, genetics, evolution, ecology, and human body systems).[web:2]
- Exam-style format to simulate real test conditions.
- Instant access via a simple, lightweight web page.

## How to Use Locally

1. Clone or download this repository.  
2. Open the `index.html` file in any modern web browser.  
3. Complete the mock exam and use it to gauge which topics to review more before the final.

## Purpose

This project reflects a **proactive** approach to learning by combining basic web development with biology content to create a practical study tool for classmates and personal exam preparation.[web:4]
